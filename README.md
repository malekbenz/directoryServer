# directoryServer

## Application that serve a direcory from web page


##Directory to share 
1. You can share a folder edit routes/browseRouter.js
2. Modify var sharedPath ='Y:\\' ;  to  your directory name 
ex: 
    var sharedPath ='D:\\' to share d:\ directory on windows 

## Run the app :
 1. npm install 
 2. bower install
 3. node server
